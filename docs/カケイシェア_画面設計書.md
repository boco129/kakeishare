## 1. 画面一覧とナビゲーション構成

### 下部タブバー（4タブ）

| タブ | アイコン | 画面名 | 説明 |
|------|---------|--------|------|
| ホーム | 🏠 Home | ダッシュボード | 残予算・今月の支出・アラート表示 |
| 支出 | 📋 List | 支出管理 | 支出一覧・CSV取り込み・手入力 |
| レビュー | 📊 Chart | 家計レビュー | グラフ・月次レポート・夫婦比較 |
| 設定 | ⚙️ Settings | 設定 | 予算・カテゴリ・分割払い・アカウント |

### 画面遷移図

```
[ダッシュボード]
  ├─ タップ: 残予算 → [予算設定]
  ├─ タップ: 分割払い → [分割払い管理]
  ├─ タップ: CSV未取り込み → [CSV取り込み]
  └─ タップ: 未確認支出 → [支出一覧（未確認フィルター）]

[支出管理]
  ├─ [支出一覧]（デフォルト表示）
  │   └─ タップ: 各明細 → [支出詳細・編集]
  ├─ FAB(+): → [手入力フォーム]
  └─ ヘッダーボタン: CSV取込 → [CSV取り込みウィザード]
      ├─ Step1: ファイル選択
      ├─ Step2: カード会社選択
      ├─ Step3: プレビュー・確認
      └─ Step4: 完了 → [支出一覧（未確認フィルター）]

[家計レビュー]
  ├─ [月次サマリー]（デフォルト）
  ├─ [カテゴリ別分析]
  ├─ [月次推移グラフ]
  └─ [Claudeレポート]

[設定]
  ├─ [予算設定]
  ├─ [カテゴリ管理]（プライバシーレベル設定含む）
  ├─ [分割払い管理]
  ├─ [カード会社・CSVマッピング]
  ├─ [LINE通知設定]
  ├─ [Claudeアドバイザー（チャット）]
  └─ [アカウント]
```

### 2.1 ダッシュボード（ホーム）

**目的**: アプリを開いた瞬間に家計の健全性がわかる
**利用者**: 夫（毎日）、妻（時々）
**デザイン方針**: 情報を絞り、数字を大きく。一目で「安心 or 危険」が伝わる

#### レイアウト（上から順に）

```
┌─────────────────────────────┐
│ ヘッダー: 2月の家計          │ ← 月切替ボタン（< 2月 >）
├─────────────────────────────┤
│                              │
│  ┌───────────────────────┐  │
│  │  残り予算              │  │ ← メインカード
│  │  ¥128,500             │  │    大きなフォント
│  │  ████████░░░ 68%      │  │    プログレスバー（青→黄→赤）
│  │  予算 ¥400,000        │  │    小さめフォント
│  │  使用 ¥271,500        │  │
│  └───────────────────────┘  │
│                              │
│  ┌──────────┐ ┌──────────┐  │ ← 2カラム情報カード
│  │ 今月支出  │ │ 先月比   │  │
│  │ ¥271,500 │ │ +¥12,000 │  │    先月比は赤(増)/緑(減)
│  │          │ │ ↑ 4.6%   │  │
│  └──────────┘ └──────────┘  │
│                              │
│  ⚠️ アラートエリア            │ ← 条件付き表示
│  ┌───────────────────────┐  │
│  │ 🔴 妻のCSV: 2月未取込  │  │    CSV未取り込み
│  │ 🟡 食費が予算の85%到達  │  │    予算超過警告
│  │ 📋 未確認の支出が12件   │  │    未確認支出
│  └───────────────────────┘  │
│                              │
│  分割払い残高                 │ ← 残債がある場合のみ表示
│  ┌───────────────────────┐  │
│  │ 3件の分割払い           │  │
│  │ 月々 ¥24,667           │  │    月々の支払総額
│  │ 残債合計 ¥148,000      │  │    残債総額（警告色）
│  │              詳細 →    │  │
│  └───────────────────────┘  │
│                              │
├─────────────────────────────┤
│ 🏠   📋    📊    ⚙️       │ ← 下部タブバー
└─────────────────────────────┘
```

#### コンポーネント仕様

| 要素 | 挙動 |
|------|------|
| 残予算カード | 予算未設定時は「予算を設定してください」のCTA表示 |
| プログレスバー | 0-60%: 青、60-80%: 黄、80-100%: オレンジ、100%超: 赤 |
| 先月比カード | 前月同日時点の支出と比較。データ不足時は「-」表示 |
| アラートエリア | アラートがなければ非表示（スペース節約） |
| 分割払いカード | 分割払いが0件なら非表示 |

## 3. プライバシー表示ルール

### 相手の支出を閲覧する場合の表示変換

| 公開レベル | 一覧での表示 | 詳細画面 |
|-----------|------------|---------|
| PUBLIC（公開） | 日付・店舗名・金額・カテゴリ全て表示 | すべて閲覧可（編集不可） |
| AMOUNT_ONLY（金額のみ） | 日付・金額・カテゴリ表示、店舗名は「個人支出」 | 店舗名・メモは非表示 |
| CATEGORY_TOTAL（合計のみ） | 一覧に個別明細を表示しない | アクセス不可 |

### CATEGORY_TOTAL の表示

支出一覧の最下部またはサマリーエリアに:
```
──── 非公開カテゴリ ────
個人娯楽    ¥25,000（3件）
```
